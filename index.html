  • Emmet

    Как мне не хватало умения пользоваться аббрeвиатурами программы Emmet и сниппетами, этого не передать.

    Советую освоить каждому. Вся эта новость пишется в разметке в 6 строк. Не делайте так. Поразительно удобно, если хочется уменьшить количество монотонной работы. Однако в VS Code без допиливания не работает, во всяком случае у меня, так что делюсь ссылкой на сработавший фикс.

    Алгоритм прост:

    1. В консоли вводим code /c/Users/место для имени вашего пользователя/AppData/Roaming/Code/User/settings.json
      или
      Находим в проводнике файл конфигурации, для этого вставляем путь до папки из команды выше, не забывая править имя пользователя.

    2. В файл добавляем строку
      {"emmet.triggerExpansionOnTab": true,}

    Ссылка на семантику аббревиатур.

    P.S. Если решите "сделать по красоте", вставить новую строчку в уже имеющиеся в файле конфигураций фигурные скобки, не забудьте про запятую между правилами. Это сэкономит вам время... Семантика - важно.
